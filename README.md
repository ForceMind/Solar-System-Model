# Solar System Model - Deployment Build

This directory contains the production build of the Solar System Model.

- **Live Site**: Open `index.html` in your browser or serve this directory.
- **Source Code**: The source code is located in the `source_code/` directory.

## Development

To continue development:

1.  Navigate to the source directory:
    ```bash
    cd source_code
    ```

2.  Install dependencies (if needed):
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  To build a new version:
    ```bash
    npm run build
    ```
    Then copy the contents of `dist/` to the root directory manually.
